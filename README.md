# off-thread Commerce

A complete local fashion e-commerce storefront built with React, TypeScript, HTML, CSS, and JavaScript.

## Features

- UI/UX redesigned from the supplied fashion storefront reference with an editorial hero, mobile category tabs, product shelves, and bottom app navigation.
- Generated editorial model hero asset based on the second supplied image as a loose model reference, with changed outfit, background, and campaign style.
- Product catalog with search, category tabs, sorting, product quick view, variants, and wishlist.
- Persistent cart with quantity controls, line removal, promo code `STYLE10`, tax, shipping, and order totals.
- Sign in and create account modal saved in browser storage.
- Ethereum wallet connection through `window.ethereum` for MetaMask or compatible wallets.
- Checkout with delivery form, shipping speed, payment method, order confirmation, and saved recent orders.
- Responsive UI for desktop, tablet, and mobile screens.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
```
